SmartFixer Application
====================

Thank you for downloading SmartFixer!

This is a standalone application that includes everything needed to run SmartFixer on your computer.

To use the application:

1. Double-click "SmartFixer-App.exe" to start the application
   OR
   Double-click "Run-SmartFixer-App.bat" to run with a startup message

2. The application will automatically open in your default web browser at:
   http://localhost:5000

3. To stop the application, close the command prompt window that appears

Features:
- No installation required
- No Python needed on your system
- Single executable file
- Custom application icon
- Runs in the background
- Automatically opens in your browser

System Requirements:
- Windows 7 or higher
- Internet connection (for initial setup)

Troubleshooting:
- If the browser doesn't open automatically, manually navigate to http://localhost:5000
- If you get a security warning, click "More info" and then "Run anyway"
- Make sure port 5000 is available on your system

Enjoy using SmartFixer!
